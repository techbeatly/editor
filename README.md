### Tips for Contributors
* **Tone:** Professional yet conversational. Think of it as a "helpful peer" sharing knowledge.
* **Images:** Place images in `/static/images/tb-uploads/YYYY/MM/` and reference them starting from `/images/...`.
* **Slug:** Keep the `url` short and lowercase, using hyphens to separate words.
* **Code Blocks:** Always specify the language (e.g., `shell`, `yaml`, `python`) for proper syntax highlighting.
